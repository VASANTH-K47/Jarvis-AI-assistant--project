# Jarvis AI Voice Assistant

## Tech Stack

### Programming Language

-   **Python**

### Libraries / Frameworks

-   **SpeechRecognition** -- Converts voice input into text.
-   **pyttsx3** -- Converts text responses into speech.
-   **Tkinter** -- Builds the graphical user interface (GUI).
-   **pywhatkit** -- Plays YouTube videos and performs web searches.
-   **Wikipedia API** -- Retrieves short summaries and information.
-   **pyjokes** -- Generates jokes for user interaction.

### Core Technologies

-   **Speech Recognition (Voice to Text)**
-   **Text-to-Speech (TTS)**
-   **Python Automation**

### Tools & Environment

-   **Visual Studio Code** -- Development environment.
-   **Git** -- Version control system.
-   **GitHub** -- Project hosting and collaboration platform.

## Project Description

The Jarvis AI Voice Assistant is a Python-based desktop assistant that
allows users to interact with their computer using voice commands. It
listens to user input, converts speech into text, processes the command,
and performs actions such as playing music, checking the time and date,
opening websites, retrieving information, and executing system tasks.

## Features

-   Voice-based command execution
-   Text-to-speech responses
-   Wake word activation ("Hey Jarvis")
-   GUI interface with voice wave animation
-   System automation (open apps, shutdown, etc.)

## Future Enhancements
